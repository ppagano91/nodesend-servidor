# Backend NodeSend

## Instalar Dependencias

```bash
npm install
```

## Variables de Entorno

Crear `.env`
```bash
cp .env.example .env
```

Completar los valores de las variables de entorno
```bash
FRONTEND_URL=http://localhost:3000
PORT=4000
DB_URL=mongodb://<host_db>:<port_db>/<database_name>
SECRETA=clave_secreta
```
